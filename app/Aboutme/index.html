หลักการและเหตุผล

เนื่องจากปัจจุบันผู้คนนิยมดื่มกาแฟกันอย่างแพร่หลาย อีกทั้งผู้คนส่วนใหญ่ในสมัยนี้ให้ความสนใจในเรื่องของกาแฟมากขึ้น ทางผู้จัดทำได้เล็งเห็นความสำคัญของกาแฟ จึงจัดทำเว็บไซต์ Coffee every daysขึ้นมา โดยจะอำนวยความสะดวกต่างๆให้ง่ายต่อการศึกษาในเรื่องของกาแฟ ซึ่งใช้งานได้ง่าย สะดวก รวดเร็ว เป็นการให้ความรู้ทางออนไลน์ทางหนึ่งซึ่งเป็นตัวแปรหนึ่งที่ช่วยในการตัดสินใจของผู้ใช้ในด้านความสะดวกในการศึกษาเรื่องกาแฟ เช่น การศึกษาข้อมูลผ่านทางเว็บไซต์ ทำให้ผู้ใช้รู้สึกว่ามีความสะดวกและอัพเดทข่าวสารได้รวดเร็วมากขึ้น

เว็บไซต์ Coffee every days เป็นเว็บไซต์ที่รวบรวมข้อมูลข่าวสารต่างๆเกี่ยวกับประโยชน์ของกาแฟ สายพันธุ์กาแฟ สูตรการชงกาแฟหลากหลายเมนู ข้อสงสัยต่างๆเกี่ยวกับกาแฟ เช่น ทำไมกาแฟสำเร็จรูป ไม่อร่อยเท่ากาแฟสด , กาแฟไม่ใส่น้ำตาลก็หวานได้ เป็นต้น โดยผู้จัดทำได้เลือกใช้โปรแกรม Visual Studio Code มาเป็นเครื่องมือในการเขียน Code เพราะใช้งานง่าย และมี Extensions ให้ดาวน์โหลดและติดตั้งมากมาย อีกทั้งยังใช้ Bootstrap CSS Framework มาเป็นตัวช่วยในการตกแต่งเว็บไซต์ให้สวยงามยิ่งขึ้น แต่ในปัจจุบันเว็บไซต์ได้รับความนิยมลดลง ผู้คนไม่นิยมเข้าเว็บไซต์มากนัก เมื่อเปรียบเทียบกับ Social Media อื่นๆ เช่น Line Facebook Instagram Twitter โดยผู้จัดทำได้เล็งเห็นถึงปัญหาดังกล่าว จึงได้นำ Line Chatbot มาใช้ให้เกิดประโยชน์ต่อเว็บไซต์ Coffee every days โดยการสร้าง Line Chatbot ผู้จัดทำได้เลือกใช้ Dialogflow เป็น AI ในการตอบคำถาม เพื่อสะดวกและเป็นที่น่าสนใจของผู้เข้าชมมากยิ่งขึ้น

จากเหตุผลดังกล่าวคณะผู้จัดทำจึงสร้างเว็บไซต์ Coffee every day ขึ้นมาและเล็งเห็นว่าจะเกิดประโยชน์ต่อผู้ที่ชื่นชอบการดื่มกาแฟและผู้สนใจทั่วไป ได้ใช้เว็บไซต์นี้เป็นการแลกเปลี่ยนข้อมูลการเรียนรู้ได้อีกช่องทางหนึ่ง

วัตถุประสงค์

  1. เพื่อแนะนำผู้เข้าชมดื่มกาแฟอย่างถูกวิธี
  2. เพื่อแนะนำสูตรการชงกาแฟแก่ผู้เข้าชม
  3. เพื่อให้สะดวกต่อการเข้าชมเว็บไซต์

เป้าหมายของการดำเนินโครงการ

  1. ให้ผู้ที่เข้ามาชมได้ดื่มกาแฟอย่างถูกวิธี
  2. ให้ผู้เข้าชมสามารถนำสูตรการชงกาแฟไปชงดื่มในชีวิตประจำวัน หรือประกอบอาชีพได้
  3. พัฒนา Line Chatbot ให้เชื่อมโยงกับเว็บไซต์
